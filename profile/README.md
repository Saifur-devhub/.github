# Saifur-devhub

Saifur-devhub is a collaborative learning community for beginners who want to learn web development step by step.

Our main goal is to learn by practicing, solving problems together, and building real projects.

## What We Are Learning

- Git and GitHub
- HTML
- CSS
- Tailwind CSS
- JavaScript
- React.js
- Node.js
- Express.js
- MongoDB
- MERN Stack Projects

## How We Learn

We follow a simple group study system:

- Weekly topic-based timeline
- 5 Google Meet sessions per week
- Self-study from YouTube, documentation, and other resources
- Problem-solving sessions
- Practical assignments
- GitHub-based project submission

## Our Learning Rule

We do not believe in copy-paste learning.

Everyone should try first, research first, and practice first.  
If someone gets stuck, the group will help.

## Current Focus

### First 3 Weeks

- Git and GitHub
- HTML
- CSS
- Tailwind CSS

By the end of this phase, every member should be able to create and publish a basic responsive website.

## Community Rules

- Respect every member
- Ask questions without fear
- Help others when possible
- Do not mock beginner mistakes
- Submit tasks on time
- Do not copy projects without understanding
- Keep learning consistently

## Repository Rules

- Do not push directly to the main branch
- Work on your own branch
- Create a pull request
- Wait for review before merging
- Never upload `.env`, API keys, passwords, or private credentials

## Our Goal

Our long-term goal is to build strong MERN stack skills through teamwork, real projects, and open-source-style collaboration.

Learning alone is hard.  
Learning together makes the journey easier.

Let’s build, break, fix, and grow together.
